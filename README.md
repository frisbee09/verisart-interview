# verisart-interview

Coding challenge for Verisart interview

## Development

### Prerequisites

- NPM installed
- MongoDB installed

### Setup

First install yarn globally using

```
npm i -g yarn
```

From there, you can run

```
> yarn install
> mongod
> yarn setup
```

### Development server

One command will boot a hot reloading backend service and a hot reloading front end server too.

```
> mongod
> yarn start
```
